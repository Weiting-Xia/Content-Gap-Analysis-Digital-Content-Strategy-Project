# 📰 AARP Content Gap Analysis & Digital Content Strategy Project

NLP-based content analytics project comparing **AARP** and **NYT** articles to identify topic gaps and support digital content strategy recommendations.

## 🔍 Overview
Analyzed **10K+ historical content records over 3 years** using Python, Databricks, and BERTopic to compare topic distributions between AARP and mainstream media.

## 🛠 Tech Stack
Python · Databricks · BERTopic · Pandas · Seaborn · NLP

## 📊 Key Findings
Found major content gaps in:
- 🏛 Politics
- 🎬 Entertainment

## 💡 Recommendations
Proposed recommendations from:
- ⏱ Timing perspective
- 🔍 Similarity perspective
- 🧪 A/B testing perspective

## 🚀 Outcome
Received positive client feedback and was selected as a **Top 3 Team**.
